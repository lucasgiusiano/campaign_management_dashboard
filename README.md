# Campaign Dashboard

A React-based campaign management dashboard built with Vite.

## Features

- View all campaigns in a sortable table
- Add new campaigns via a form with validation
- Delete campaigns with confirmation step
- Dynamic profit calculation (Revenue - Cost)
- Summary cards (total campaigns, cost, revenue, profit)
- Sort by name, start date, or profit
- Data persisted in localStorage
- Live profit preview while filling the form

## Stack

- React 18
- Vite 4

## Getting Started

```bash
npm install
npm run dev
```

Then open http://localhost:5173
